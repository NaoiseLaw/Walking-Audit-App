## 🎯 Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
npm install
cd backend && npm install
cd ../frontend && npm install
```

### 2. Set Up Environment
```bash
# Backend
cd backend
cp .env.example .env
# Edit .env with your configuration

# Frontend
cd frontend
cp .env.example .env
# Edit .env with your configuration
```

### 3. Set Up Database
```bash
cd backend
npm run db:migrate
npm run db:seed
```

### 4. Start Servers
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm run dev
```

### 5. Access Application
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:3001
